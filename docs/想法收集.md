### 3. AIOEP 作为独立主项目，ERP 是验证实例

**记录时间**：2026-02-16

**核心认知**：
- AIOEP（AI 组织效能平台）是主项目，不是 ERP 的附属
- ERP 项目是 AIOEP 平台的第一个验证实例
- AIOEP 需要分模块开发，每个模块完成后用 ERP 来验证
- 需要独立的项目仓库、战略规划、模块拆分和路线图

**AIOEP 可能的模块拆分**：
- 方法论引擎（AI-EADM 9 域）
- 方法库（分析/开发/交付方法 + 模板）
- Skills 引擎（L1/L2/L3 三级管理）
- 项目上下文管理（Phase 追踪、状态管理）
- 工具集成（Cursor Rules/Skills 生成、Archi/draw.io）
- 多项目管理
- Web 看板（后期）

**开发验证循环**：
```
开发 AIOEP 模块 N → 用 ERP 项目验证模块 N → 回顾改进 → 开发模块 N+1
```

**下一步**：新对话中做 AIOEP 的战略规划和模块规划
