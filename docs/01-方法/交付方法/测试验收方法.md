# 测试验收方法

> 状态: **纲要** | 版本: v0.1.0 | 创建: 2026-02-16 | 上次验证: 未验证

## 定位

指导如何执行端到端集成测试、UAT 验收测试、性能测试和安全审计。

## 前置条件

- Phase 2 退出条件全部满足（所有模块开发完成）
- 测试环境（staging）准备就绪
- BDD 行为规格可用（作为 UAT 测试用例来源）

## 步骤

1. 集成测试：跨模块端到端业务流程验证
2. UAT 验收：业务方按 BDD 场景逐项确认
3. 性能测试：核心页面响应时间 + 并发用户测试
4. 安全审计：权限验证 + 数据安全检查

> 各步骤的详细执行指南待细化。

## 检查清单

- [ ] 集成测试覆盖所有端到端流程
- [ ] UAT 业务方签字确认
- [ ] 性能测试响应时间达标
- [ ] 安全审计无 P0/P1 漏洞

## 输出

- UAT 测试报告（签字）
- 性能测试报告
